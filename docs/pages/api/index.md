## Geolonia の JavaScript API について
Geolonia の JavaScript API は、Mapbox GL JS の拡張クラスで、同じオプションやプロパティ、メソッド等が使用できます。

また、Embed API によって、緯度や経度などのオプションを HTML の data-* 属性を使用して設定することができ、少ない記述で地図アプリケーションの開発を始めることができます。
